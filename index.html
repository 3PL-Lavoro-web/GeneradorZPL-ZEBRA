Cómo enviar a la ZT411
  1. Abre Zebra Setup Utilities → selecciona la impresora → Open Communication With Printer.
  2. File → Open → elige el .zpl descargado (filtro "All files").
  3. Clic en Send To Printer. Prueba siempre primero con la Muestra (5).
  4. Por red también sirve: curl --data-binary @archivo.zpl http://IP_IMPRESORA:9100
  5. Si la impresión queda desfasada, autocalibra: mantén PAUSE + FEED en el panel.
